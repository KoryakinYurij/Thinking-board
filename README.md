# To Do Board

Task-first To Do app with a disciplined kanban projection.

## Stack

- React
- TypeScript
- Vite

## Product stance

- `To Do app first`
- `Kanban second`
- completion must be easier than drag-and-drop
- board columns are a projection of task status

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
npm test
```
